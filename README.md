# Proxima Chat — Open WebUI Custom

Image Docker legere basee sur [Open WebUI](https://github.com/open-webui/open-webui) avec le branding Proxima Green.

## Deploiement rapide (Coolify / Docker)

```bash
docker compose up -d
```

Variables d'environnement : voir `.env.example`

## Ce que le branding change

- Logo & favicon Proxima (deux fleches vertes)
- Palette teal-green (`#143330`, `#2d6b66`)
- Nom "Proxima Chat" partout
- Suppression du suffixe "(Open WebUI)"
- PWA manifest Proxima

Les fonctionnalites Open WebUI restent 100% intactes.
